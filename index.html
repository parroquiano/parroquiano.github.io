Personales y colaborativos

DO DynDNS Cliente simple de DNS dinámico para DigitalOcean.

Para actualizar uno o más registros DNS (para dominios gestionados por DigitalOcean) con la dirección IP pública de un host cliente.
cPanel CLI Una interfaz de línea de comandos para la Unrestricted API (UAPI) de cPanel.

Ejecute comandos rápidamente en una terminal para gestionar un sitio web, evitando la necesidad de iniciar sesión en cPanel y navegar por su interfaz web. Este utilitario también se puede integrar sin problemas en tareas de scripting.
cPanel API Fork de tz4678/cpanel-api. Cliente de API de cPanel para Python.

Agregué una opción de autenticación con tokens UAPI. Lo uso para cPanel CLI.
IMAP Copy Fork de r00tat/imap_copy. Un utilitario simple para copiar carpetas de correos entre servidores IMAP.

Hice este fork para agregar una opción para examinar la conexión IMAP y mostrar una lista de las carpetas remotas. Además corregí un par de bugs que encontré para Hotmail y Gmail. Este utilitario lo usé para para migrar diez años de mensajes de correo cuando cambié de proveedor de correo electrónico.
