todoister list

todoister list [project]...

Lista proyectos y subproyectos.

project... son los nombres de uno o más proyectos o subproyectos a listar. Si no se proporciona un nombre de proyecto, se listarán todos los proyectos. Puede especificar un nombre de proyecto por su ruta completa, por ejemplo, "Work/Project". Los nombres no distinguen entre mayúsculas y minúsculas.

Opciones comunes

-t, --token <string>
Reemplazar el token de Todoist

Ejemplos

todoister ls
todoister ls Work Life
todoister ls Work/Project

todoister export

todoister export [path] [flags]

Exporte todos los proyectos de Todoist.

  • path archivo o directorio donde exportar los proyectos, por defecto index.json o index.yaml (si se especifica --yaml) en el directorio actual.

  • -d, --depth N cuando se proporciona, todoister creará directorios hasta N niveles de profundidad, escribiendo cada subproyecto en un archivo separado.

Opciones

-d, --depth <int>
Profundidad del árbol de subdirectorios al exportar (por defecto 0, es decir, sin subdirectorios)
--json
Exportar en formato JSON (por defecto)
--yaml
Exportar en formato YAML

Opciones comunes

-t, --token <string>
Sobrescribir el token de Todoist

Ejemplos

todoister export
todoister export ~/todoist.json
todoister export ~/todoist.yaml --yaml
todoister export ~/projects --json -d 3
